# 3D_Prints
Repo for 3D Models.

## Usage

Pick one:

    git clone --recurse-submodules https://github.com/jed-frey/3D_Prints.git

    git clone --recurse-submodules git@github.com:jed-frey/3D_Prints.git

Change directory:

    cd 3D_Prints/
    
Update submodules. Make .gcode from .stl.
    
    make update
    make -j8
